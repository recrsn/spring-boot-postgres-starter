# Spring Boot + PostgreSQL starter

Starter project with Java 11, Spring Boot v2.4.3 and Postgres13+

Contains:

- Java 11
- Gradle
- Devtools
- Actuator
- Lombok

## Getting Started

### Requirements

- JDK 11
- Docker

## Running the project

1. Start the included PostgreSQL 13 server with `docker compose up`
2. Start Spring Boot with `./gradlew bootRun`
